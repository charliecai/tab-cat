# Tab Cat Store Listing Copy

Author: charliec

## English Listing

### Extension Name

Tab Cat

### Short Description

Keep tabs on your tabs with a calm new tab dashboard, duplicate cleanup, pinned shortcuts, and a local reading inbox.

### Full Description

Tab Cat replaces your new tab page with a clean dashboard for everything you already have open.

Instead of digging through crowded browser windows, you get a grouped view of your open tabs, organized by domain. Common homepage-style tabs such as Gmail, X, LinkedIn, YouTube, and GitHub can be treated as their own cleanup lane, so you can quickly see what is useful and what is just browser clutter.

What Tab Cat helps you do:

- See open tabs grouped by domain in one dashboard.
- Find duplicate tabs and close extra copies.
- Click any tab title to jump directly to the existing tab.
- Close one tab or a whole group when you are done with it.
- Keep a small set of pinned shortcuts above your live tab grid.
- Save article-like pages into a Reading inbox before closing the tab.
- Review saved reading items later, mark them read, archive them, or delete them.
- Optionally analyze saved articles with your own OpenAI-compatible AI provider.

Tab Cat is local-first. There is no Tab Cat account and no Tab Cat server. Core browsing state is stored in your browser profile using extension storage and IndexedDB.

AI analysis is optional. If you choose to configure an OpenAI-compatible provider, captured article text is sent directly from your browser to the HTTPS provider host you enter in Settings. If you do not configure AI, the core tab dashboard, duplicate cleanup, pinned shortcuts, and reading inbox still work locally.

Tab Cat is built for people who keep too many tabs open but still want their browser to feel calm, fast, and under control.

Tab Cat is derived from the open-source project at https://github.com/zarazhangrui/tab-out.

## Chinese Listing

### 扩展名称

Tab Cat

### 简短描述

把新标签页变成安静的标签页仪表盘，支持按域名分组、重复页清理、固定入口和本地阅读收件箱。

### 完整描述

Tab Cat 会把你的浏览器新标签页替换成一个清爽的标签页仪表盘，让你一眼看清现在到底开了什么。

它会按域名整理当前打开的标签页，把分散在多个窗口里的页面集中展示。Gmail、X、LinkedIn、YouTube、GitHub 这类常见首页也可以被归到更容易清理的位置，帮你区分哪些页面还要用，哪些只是浏览器里越积越多的杂物。

Tab Cat 可以帮你：

- 在一个仪表盘里按域名查看所有打开的标签页。
- 发现重复打开的页面，并快速关闭多余副本。
- 点击标签页标题，直接跳回已经打开的页面。
- 单独关闭一个标签页，或一次关闭某个分组。
- 在标签页网格上方固定少量常用入口。
- 把文章型页面保存到 Reading inbox，再放心关闭原标签页。
- 稍后查看已保存文章，并标记已读、归档或删除。
- 可选接入你自己的 OpenAI 兼容 AI 服务，对保存文章做分析。

Tab Cat 是本地优先的扩展。它不需要 Tab Cat 账号，也没有 Tab Cat 服务端。核心浏览状态会保存在你的浏览器配置中，使用扩展存储和 IndexedDB。

AI 分析是可选功能。只有当你在 Settings 中配置 OpenAI 兼容服务后，Tab Cat 才会把抓取到的文章正文直接从浏览器发送到你填写的 HTTPS provider host。不配置 AI 时，标签页仪表盘、重复页清理、固定入口和阅读收件箱依然可以在本地正常使用。

Tab Cat 适合经常打开很多标签页、但又希望浏览器保持清爽和可控的人。

Tab Cat 衍生自开源项目：https://github.com/zarazhangrui/tab-out。
